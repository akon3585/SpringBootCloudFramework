---
title: LCDComponent
subtitle: 模拟LCD液晶显示字体
---

目前仅支持数字、字母和部分字符[.:-/%]

## API

| 参数  | 说明                                | 类型   | 默认值    |
| ----- | ----------------------------------- | ------ | --------- |
| size  | 液晶字体的大小(default,small,large) | string | 'defalut' |
| color | 液晶字体的颜色                      | string | '#000000' |

## demo

```js
<LCDComponent size='' color='#00ffff'>
    2019-04-24 12:44:59
</LCDComponent>
```
